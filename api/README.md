# Aplicação DevBook

## API DevBook

### Comandos básicos para Golang

### Criar um módulo

`go mod init api`

### Atualizar o módulo da aplicação com os novos imports

`$ go mod tidy`

### Rodar a aplicação

`$ go run main.go`

### Gerar build da aplicação(arquivo executável)

`$ go build`

### Instalação de módulos
